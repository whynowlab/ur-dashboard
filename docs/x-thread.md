### Tweet 3/7 — Solution
ur-dashboard gives you live visibility into your Claude Code workflow.

```bash
npm install -g ur-dashboard
ur-dashboard
```

Then open `localhost:3000`.

No complex setup. Just run it and see what your agents are doing.
